# AnimeList

Современный сайт для отслеживания аниме, созданный с использованием Next.js и Jikan API.

## Функциональность

- Просмотр популярных аниме
- Поиск аниме
- Детальная информация об аниме
- Список персонажей
- Отслеживание прогресса просмотра
- Оценка аниме
- Профиль пользователя

## Технологии

- Next.js
- TypeScript
- CSS Modules
- Jikan API

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/animelist.git
cd animelist
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите проект в режиме разработки:
```bash
npm run dev
```

4. Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере.

## Сборка для продакшена

```bash
npm run build
npm start
```
